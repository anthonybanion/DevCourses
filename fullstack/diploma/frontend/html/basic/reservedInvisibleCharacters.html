Reserved or Invisible Characters Table

HTML entities are used to represent reserved or invisible characters in HTML. Here is a comprehensive table grouped by categories.

1. Basic Symbols

Entity Result Meaning Example
&amp; & Ampersand &amp;copy; → ©
&lt; < Less than <p>
&gt; > Greater than </p>
&quot; " Double quote "text"
&apos; ' Single quote (apostrophe) It's fine
&nbsp;   Non-breaking space word word

2. Currency Symbols

Entity Result Meaning Example
&euro; € Euro Price: €100
&cent; ¢ Cent 50&cent;
&pound; £ Pound Amount: £25
&yen; ¥ Yen Cost: ¥1000
&dollar; $ Dollar (use $ or just $) Price: $10

3. Mathematical Symbols

Entity Result Meaning Example
&plusmn; ± Plus-minus x ± y
&times; × Multiplication 3 × 4 = 12
&divide; ÷ Division 10 ÷ 2 = 5
&le; ≤ Less than or equal x ≤ y
&ge; ≥ Greater than or equal x ≥ y
&ne; ≠ Not equal a ≠ b

4. Greek Letters

Entity Result Letter Example
&alpha; α Alpha α = a
&beta; β Beta β = b
&gamma; γ Gamma γ = g
&delta; δ Delta δ = d
&lambda; λ Lambda λ = wavelength
&pi; π Pi π ≈ 3.14
&sigma; σ Sigma σ = sum
&omega; ω Omega ω = resistance

5. Arrows

Entity Result Direction Example
&larr; ← Left ← Back
&rarr; → Right Next →
&uarr; ↑ Up Top ↑
&darr; ↓ Down Scroll ↓
&harr; ↔ Left-Right Swap ↔

6. Logical and Miscellaneous Symbols

Entity Result Meaning Example
&and; ∧ Logical AND A ∧ B
&or; ∨ Logical OR A ∨ B
&not; ¬ Logical NOT ¬A
&infin; ∞ Infinity ∞ loop
&asymp; ≈ Approximately equal π ≈ 3.14
&deg; ° Degree 90°
